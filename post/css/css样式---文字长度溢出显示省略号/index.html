CSS样式----文字长度溢出显示省略号

发表于： 2016-11-02 分类于： Web
字数：312 阅读：≈ 1分钟 浏览：

text-overflow省略号样式语法结构

  • text-overflow语法： text-overflow : clip | ellipsis

  • text-overflow参数值和解释： clip : 　不显示省略标记（…），而是简单的裁切 ellipsis : 　当对象内文本溢出时显示省略标记（…）

  • text-overflow应用说明： CSS text-overflow设置或检索是否使用一个省略号标记（…）标示对象内文本文字的溢出。 要想实现显示不完内容将显示省略号代替，还需要html nobr标签完成（nobr禁止换行标签）

text-overflow应用案例

  • 1、实现方法 对象设置text-overflow:ellipsis;省略号样式
width:280px;
display: inline-block;
text-overflow:ellipsis; 
white-space:nowrap; /*规定段落中的文本不进行换行*/
overflow:hidden;
